# Gestion de buvettes associatives

Ceci est un projet universitaire réalisé en **2e année de BUT Informatique**.  
Il s'agit d'un site web permettant la **création et la gestion de buvettes associatives**, développé en **PHP selon une architecture MVC**.

## Contributeurs

- Wael BEN MOUSSA
- Lino COUTURIER

## Fonctionnalités

- Création et gestion de buvettes associatives
- Gestion des utilisateurs
- Ajout, modification et suppression de produits
- Gestion des stocks

## Technologies utilisées

- PHP
- HTML / CSS
- MySQL
- Architecture MVC (Modèle-Vue-Contrôleur)

## Installation et exécution

Le projet est destiné à être exécuté sur un **serveur local compatible PHP et MySQL**
(exemples : XAMPP, WAMP).

Le dossier `freshbreak/` correspond à la **racine de l’application web**.

Pour des instructions détaillées concernant :
- l’installation,
- la configuration de la base de données,
- et le lancement du projet,

veuillez [consulter le guide dédié (HOWTO.md)](./HOWTO.md)
